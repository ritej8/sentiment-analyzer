📝 Sentiment Analyzer

Sentiment Analyzer is a deep learning-based web application that predicts the sentiment (positive or negative) of user-provided text. The project demonstrates a complete NLP pipeline using PyTorch and a Flask API for deployment.

⚡ Features

Text Preprocessing & Vectorization: Converts raw text into numerical vectors using CountVectorizer.

Neural Network Model: Simple feedforward PyTorch neural network for sentiment classification.

Prediction API: Flask API to send text and receive predicted sentiment.

Trained Model & Pipeline Saved: Model weights (.pth) and vectorizer (.pkl) stored for reproducible predictions.

Deployment Ready: Can be extended for web or mobile frontend integration.

🛠️ Tech Stack

Language: Python

Frameworks & Libraries: PyTorch, scikit-learn, Flask, Joblib, Pandas, NumPy
